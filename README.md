# project-boilerplates

多合一工程项目样板集合 / All-in-one project boilerplates

## 它是什么

旨在成为一个工程 cli 入口，方便开发

<p > 
    <img width="400" src="./project-boilerplates.gif"> 
</p>

## 下载

```bash

yarn add global project-boilerplates

# 或者

npm install -g project-boilerplates

```

## 使用

```bash

pb init

```
